Multi-tenant single page app container for Node.js and Express.

Multi-Tenant

With a single 4front instance, host many virtual web apps as you like, each with a unique URL like myapp.hostcontainer.com.

Single Page Apps

Purpose built for today's single page applications built with tools such as AngularJS, Ember, and React.

Performance

4front adds an additional thin layer atop Express providing the minimal foundation required for multi-tenant hosting of single page apps.

Philosophy

Carry on the Express philosophy of a fast, un-opinionated, and highly modular web framework. Promotes the tenants of 12 factor apps.
