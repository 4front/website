Multi-tenant web app container for Node.js and Express.
$ npm install 4front-apphost --save

Multi-Tenant

With a single 4front instance, host many virtual web apps as you like, each with a unique URL like myapp.hostcontainer.com.

Powerful Flexibility

Purpose built for today's single page applications, but works equally well for traditional websites. Layer on authentication, advanced integration, and more.

Performance

4front adds an additional thin layer atop Express providing the minimal foundation required for multi-tenant hosting of web apps.

Philosophy

Carry on the Express philosophy of a fast, un-opinionated, and highly modular web framework. Promotes the tenants of 12 factor apps.
